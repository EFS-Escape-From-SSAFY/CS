## 역, 이, 대우

- 조건명제에서 사용

- 하나의 명제를 변형하여 표현

  

  ### 역

  - 명제의 위치를 바꿈

  ### 이

  - 두 명제 각각을 부정

  ### 대우

  - 역 + 이

  - 위치 바꾸고 두 명제 각각을 부정

    

  ### 왜?

  - 어떠한 명제를 증명하기 위함

  

  ### 진리표
